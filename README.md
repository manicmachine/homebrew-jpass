# Manicmachine Jpass

## How do I install these formulae?

`brew install manicmachine/jpass/<formula>`

Or `brew tap manicmachine/jpass` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "manicmachine/jpass"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
